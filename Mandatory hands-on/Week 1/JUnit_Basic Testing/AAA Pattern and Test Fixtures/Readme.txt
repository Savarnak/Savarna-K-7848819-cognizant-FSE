EXERCISE 4 - AAA PATTERN, SETUP AND TEARDOWN

Objective:
To demonstrate the Arrange-Act-Assert (AAA) testing pattern and use of setup and teardown methods.

Annotations Used:
1. @Before - Executes before each test method.
2. @Test - Executes the test case.
3. @After - Executes after each test method.

AAA Pattern:
Arrange - Prepare test data.
Act - Execute the method being tested.
Assert - Verify the expected result.

Result:
The test executed successfully using setup and teardown methods.
