EXERCISE 1 - SETTING UP JUNIT

Objective:
To set up JUnit in a Java project and create a basic unit test case.

Steps Performed:
1. Created a Maven Java project.
2. Added JUnit 4.13.2 dependency in pom.xml.
3. Created a TemperatureConverter class.
4. Created a JUnit test class named TemperatureConverterTest.
5. Used the @Test annotation to define a test method.
6. Used assertEquals() to verify the correctness of the temperature conversion.
7. Executed the test successfully.

Files Included:
- pom.xml
- TemperatureConverter.java
- TemperatureConverterTest.java

Expected Result:
The test case should pass successfully without any failures or errors.

Conclusion:
JUnit helps automate testing and ensures that application functionality works as expected.
