EXERCISE 3 - ASSERTIONS IN JUNIT

Objective:
To demonstrate the use of various JUnit assertions.

Assertions Used:
1. assertEquals() - Verifies expected and actual values are equal.
2. assertTrue() - Verifies a condition is true.
3. assertFalse() - Verifies a condition is false.
4. assertNull() - Verifies an object reference is null.
5. assertNotNull() - Verifies an object reference is not null.

Result:
All assertions executed successfully and the test passed.
