# Hands-on 1: Create a Spring Web Project using Maven

## Objective

Create a Spring Boot web application using Maven and verify that the application starts successfully.

## Technologies Used

- Java
- Spring Boot
- Spring Web
- Spring Boot DevTools
- Maven

## Features

- Created a Spring Boot project using Spring Initializr
- Configured Maven dependencies
- Added Spring Web and DevTools
- Verified successful application startup
- Logged the execution of the `main()` method using SLF4J

## Output

The application starts successfully, initializes the embedded Tomcat server, and logs:

```text
Started SpringLearnApplication
Inside main
```
